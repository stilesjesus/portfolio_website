# Interactive Fourier Transform Visualizer Assets

This folder contains images and media assets for the Fourier Visualizer project.

## Suggested file naming:
- `hero.jpg/png/webp` - Main project image
- `3d-view.png` - Screenshot of 3D visualization
- `2d-view.png` - Screenshot of traditional 2D view
- `controls.png` - Interface controls screenshot
- `waveforms.png` - Example waveform outputs

Add your project images here and reference them in the MDX file using:
```
/assets/projects/fourier-visualizer/filename.ext
```
